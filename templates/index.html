What is MedICI

The Medical Challenge Initiative (MedICI) is a platform to promote innovation through collaboration and harness the ingenuity of the American public. It is based on the Microsoft open-source project CodaLab, with integrations with ePad and caMicroscope. Additional customizations to enhance the project were incorporated based on user input.

CodaLab has its own installation instructions you can follow and we recommend following those. They also have a github/wiki containing instructions for how to operate the software once it is up and running. However, MedICI has a customized set of installation instructions you can perform step-by-step on Microsoft's Azure platform. These customized instructions may be all you need to install MedICI, but refer to the CodaLab documentation for supplemental information. Keep in mind when cloning this repo that you will be cloning MedICI's fork maintained by QTIM. The installation should be same as the MedICI instructions, but you will have extra code for the customizations below.

  1. CodaLab

  2. ePad is a freely available quantitative imaging informatics platform, developed at the Rubin lab. This software handles the radiology annotations and metadata management.


  3. caMicroscope, a web based annotation and visualization platform designed for digitized whole slide images. This software handles pathology annotations optimized for large bio-medical image data, with an emphasis on cancer pathology.


  4. Docker Submissions: Rather than limited to code\data submissions, users can submit whole docker images.
  5. Issue Reporting System: Participants routinely have questions that are answered in CodaLabs forums regarding first time use and the submission process. Occasionally issues with an evaluation script or website functionality prevents users from submitting their results or incorrectly scores their algorithm. MedICI provides additional functionality to track these specific issues.
